# Cypress Automation - Login E2E

Este projeto contém testes automatizados end-to-end (E2E) utilizando Cypress.

## Objetivo
Automatizar o fluxo de login de uma aplicação web, validando o acesso à área segura após autenticação com sucesso.

## Tecnologias utilizadas
- JavaScript
- Cypress
- Node.js
- Git / GitHub

## Estrutura do projeto
- cypress/e2e → testes automatizados
- cypress/support → comandos e configurações
- cypress.config.js → configurações do Cypress

## Cenários automatizados
- Acesso à página de login
- Login com credenciais válidas
- Validação da área segura após login
